# Devoir maison – Master 1 – Analyse du SMD en Python

## Contexte
Ce dépôt présente un devoir maison réalisé dans le cadre de ma première année de master. Le travail porte sur l’utilisation du Standardized Mean Difference (SMD) pour évaluer l’équilibre des covariables dans une étude observationnelle.

## Objectif
L’objectif était de comparer les groupes de patients RHC et No RHC à partir de données cliniques, de calculer les SMD pour plusieurs variables quantitatives et binaires, puis d’étudier l’effet d’une randomisation simulée du traitement sur cet équilibre.

## Méthodes
Le projet a été réalisé en Python avec :
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Analyses réalisées
- identification de variables quantitatives et binaires
- calcul des moyennes, écarts-types, effectifs et proportions par groupe
- calcul des SMD
- représentation graphique des déséquilibres
- randomisation simulée du traitement
- comparaison des SMD avant et après randomisation
